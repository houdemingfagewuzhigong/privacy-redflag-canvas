# Privacy Redflag Canvas 创新创业项目申报书

## 项目概述

Privacy Redflag Canvas 是一个面向法学生、社科研究者、记者和消费者权益教育场景的隐私政策红旗识别工具。

## 问题背景

隐私政策文本长、抽象、难比较，普通用户难以发现敏感数据收集、第三方共享、长期留存、自动化决策等风险。

## 目标用户

法学生、社会科学研究者、记者、NGO/公益组织、消费者权益教育者、大学生创新创业团队。

## 创新点

将复杂隐私政策转换成可视化红旗画布、证据表和可打印报告，降低法律科技教育和公共议题传播门槛。

## 技术路线

静态前端 + 浏览器内规则引擎 + JSON 导出 + 可打印报告 + Word 版申报材料。

## 社会价值

帮助公众理解平台隐私治理，支持法学和社科教学，辅助消费者权益倡导。

## 风险与合规

本项目不是法律建议，不替代律师或监管机构判断；所有分析均在本地浏览器完成。
